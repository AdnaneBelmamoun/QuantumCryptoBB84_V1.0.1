B92 Quantum Transmission

(responder's side)


The raw B92 quantum transmission generates the raw material from which a secret-key will be distilled. It is a physical implementation of the B92 coding scheme which is a subset of the states used in the BB84 coding scheme.

Rhough Description

A raw key is a sequence of classical bits from the which the final secret key should be distilled. A raw B92 key is a sequence of transmitted pulses that have have been detected. The B92 coding scheme encodes classical information in the bases used for the BB84 coding scheme. The pulses transmitted in the B92 coding scheme are prepared in the following quantum states:
  • For each pulse i, the bit bi to be sent is chosen randomly,
  • A qubit in state |0>+ is sent if bi=0 and a qubit in quantum state |0>X is sent if bi=1,
  • For each pulse, the receiver measures the qubit in a random basis t'i (``+'' or ``X'') and records the classical one-bit outcome b'i.

The raw key is defined as the sequence b=b1,...,bn on the sender's side and b'=b'1,...,b'n on the receiver's side for the detected pulses. In an ideal situation, the quantum transmission has the following property:
  • For all pulses i such that b'i=1 we have bi= 0 if t'i= X (diagonal) and bi=1 if t'i= + (rectilinear).
However, the quantum trasmission is not perfect and errors occur. More information on the physical implementation of the B92 coding scheme can be found here.
